## プロジェクト概要

■ 概要

本プロジェクトは、ブロックギミックを活用したC++ベースの2Dパズルアクションゲームとして開発した。
特に「レベルデザインに直結するギミック制御」の設計に重点を置いている。

■ 設計（コア部分）

・基幹システム構築（最重要）

メインゲームループ、スコアやカウントを管理するGameData、およびタイトル・ゲームクリア等のシーン遷移システムをC++で一から構築した。

・ギミック・マップ管理

BlockManagerを用いたブロック群の一括制御や、MapTipパーサー、エリアチェンジ（画面切り替え）処理を実装し、パズルアクションの基盤を確立した。

■ 実装（ゲームプレイ）

・プレイヤー制御

ジャンプや移動処理、マップとの厳密な当たり判定など、アクションゲームのコアとなるロジックを実装。

・レベルデザイン

チュートリアルからステージ10まで、徐々に難易度が上がる複数のマップとギミック配置を構築。

■ 演出・体験設計

・フィードバック制御

アクションに対するエフェクトの表示や、全ステージクリア時の特別なUI・エンドロール演出を実装。

・サウンド・ビジュアル

BGMやSEの組み込み、図形や数字を用いたスコア・カウント表示など、ゲームの雰囲気に合った視覚体験を提供。

■ 工夫・改善点

・BlockManagerを中心にギミックを管理することで、新しいステージの追加や修正を容易にした

・3名でのチーム開発において、システム基盤、レベルデザイン、バグ修正など役割を明確に分担し開発効率を上げた
